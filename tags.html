Important HTML tags

This page is designed as a quick reference for some of the of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via tables and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table header include a table data cell
div div block create a div block
header header section include a header section
ol ordered list include an ordered list
li list element include a list element
ul unordered list include an unordered list

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span group inline elements <p> My mother has <span style="color:blue">blue</span> eyes.<p>

My mother has blue eyes.
